Comment utiliser Google Tone ?

Pour diffuser une URL à l'aide de Google Tone, procédez comme suit :

  • Connectez-vous à votre compte Google.
  • Cliquez sur l'icône correspondante dans le navigateur Chrome lorsque vous consultez une page Web que vous souhaitez diffuser.

Pourquoi utiliser Google Tone ?

Cette extension permet une communication sonore entre les ordinateurs, comme lorsque nous parlons entre nous. Il s'agit d'une extension de navigateur qui permet à Chrome d'utiliser les haut-parleurs de votre ordinateur afin d'émettre une signature sonore spéciale pour qu'elle soit identifiée en tant qu'URL via le micro d'autres ordinateurs.

Comment fonctionne l'extension Google Tone ?

Lorsque l'extension Google Tone est activée, le micro de votre ordinateur est activé. Les URL sont enregistrées temporairement sur les serveurs de Google, et les haut-parleurs et le micro de votre ordinateur sont utilisés pour qu'elles soient envoyées aux ordinateurs connectés à Internet situés à proximité. Tous les ordinateurs pouvant entendre les sons émis, et sur lesquels l'extension Google Tone est installée et activée, sont susceptibles de recevoir des notifications. L'URL, ainsi que le nom et la photo de votre profil Google, s'affichent dans la notification.

Pour que vous puissiez recevoir une URL avec Google Tone dans Chrome, votre micro doit rester activé. Il est possible que Google Tone ne fonctionne pas dans les lieux bruyants, lorsque les ordinateurs sont trop éloignés les uns des autres, en cas de mauvaise connexion Internet, ou sur des ordinateurs sans micro ou dont le micro ne peut pas détecter les sons diffusés par Google Tone.

Comment mes données sont-elles utilisées pour Google Tone ?

Des données d'utilisation anonymes sont recueillies conformément aux Règles de confidentialité de Google.

Comment activer ou désactiver l'extension ?

Pour activer ou désactiver l'extension Google Tone, y compris le micro, accédez aux paramètres de l'extension Chrome.

Ce service est-il sécurisé ?

Seules des URL sont diffusées dans le cadre du service Google Tone. De cette manière, les destinataires ne peuvent pas automatiquement accéder aux pages auxquelles ils n'auraient normalement pas accès. Si vous diffusez l'URL de votre boîte de réception Gmail, par exemple, les destinataires qui cliquent sur la notification Google Tone sont invités à se connecter à leur compte Gmail. Toutefois, les messages diffusés par Google Tone sont, par définition, publics. Il est donc préférable de ne pas utiliser ce service pour échanger des informations confidentielles.
