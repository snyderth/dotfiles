Εξακολουθεί να μη λειτουργεί;

Προκειμένου να λειτουργεί σωστά, το Google Tone θα πρέπει να ακούει τον εαυτό του. Το μικρόφωνο του υπολογιστή σας ενδέχεται να είναι απενεργοποιημένο. Ελέγξτε τις ρυθμίσεις του μικροφώνου σας.

Το Google Tone υποστηρίζεται στους περισσότερους υπολογιστές, αλλά ενδέχεται να μη λειτουργεί σε ορισμένους λόγω ζητημάτων εξοπλισμού. Ενημερώστε μας εάν δεν λειτουργεί για εσάς.
