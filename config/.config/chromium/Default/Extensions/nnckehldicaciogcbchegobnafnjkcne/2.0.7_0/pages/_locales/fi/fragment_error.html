Eikö ongelma ratkennut?

Toimiakseen oikein Google Tonen on kuultava oma äänensä. Voi olla, että tietokoneesi mikrofoni on poistettu käytöstä. Tarkista mikrofoniasetukset.

Useimmat tietokoneet tukevat Google Tonea, mutta se ei välttämättä toimi kaikilla koneilla laitteisto-ongelmien vuoksi. Kerro meille, jos se ei toimi sinulla.
