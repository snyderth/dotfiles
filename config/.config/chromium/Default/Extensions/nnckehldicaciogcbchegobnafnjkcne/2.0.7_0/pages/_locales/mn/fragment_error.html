Ажиллахгүй хэвээр байна уу?

Google Tone-д дуу авиа хүрч байж, зохих ёсоор ажиллана. Таны компьютерийн микрофон унтарсан байж магадгүй тул тохиргоог нь шалгаарай.

Google Tone-ийг ихэнх компьютер дэмжиж ажилладаг ч, зарим нь техник хангамжийн асуудлаас болоод ажиллахгүй байж болно. Тиймээс ажиллахгүй бол бидэнд мэдэгдээрэй.
