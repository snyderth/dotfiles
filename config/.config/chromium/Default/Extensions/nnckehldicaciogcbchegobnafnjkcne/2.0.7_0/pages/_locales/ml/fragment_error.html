ഇപ്പോഴും പ്രവർത്തിക്കുന്നില്ലേ?

ശരിയായി പ്രവർത്തിക്കുന്നതിന്, Google ടോണിന് സ്വയം കേൾക്കേണ്ടതുണ്ട്. നിങ്ങളുടെ കമ്പ്യൂട്ടറിന്റെ മൈക്രോഫോൺ ഓഫായിരിക്കാം. നിങ്ങളുടെ മൈക്ക് ക്രമീകരണം പരിശോധിക്കുക.

Google ടോൺ മിക്ക കമ്പ്യൂട്ടറുകളിലും പിന്തുണയ്‌ക്കുമെങ്കിലും ഹാർഡ്‌വെയർ പ്രശ്‌നങ്ങൾ കാരണം ചിലതിൽ പ്രവർത്തിച്ചേക്കില്ല. അത് പ്രവർത്തിക്കുന്നില്ലെങ്കിൽ ഞങ്ങളെ അറിയിക്കുക.
