Damit die Google Tone-Erweiterung ordnungsgemäß funktionieren kann, muss sie die selbst erzeugten Tonsignale erfassen können. Möglicherweise ist das Mikrofon Ihres Computers ausgeschaltet. Überprüfen Sie Ihre Mikrofoneinstellungen.

Google Tone wird von den meisten Computern unterstützt. Auf einigen Geräten funktioniert die Erweiterung jedoch möglicherweise aufgrund von Problemen mit der Hardware nicht. Sollte dies bei Ihnen der Fall sein, informieren Sie uns bitte darüber.
