¿Aún no funciona?

Para que funcione correctamente, la extensión Google Tono necesita escucharse a sí misma. Es posible que el micrófono del equipo esté desactivado. Verifica la configuración del micrófono.

Google Tono es compatible con la mayoría de los equipos, pero es posible que no funcione en algunos de ellos debido a problemas de hardware. Si no funciona, infórmanos.
