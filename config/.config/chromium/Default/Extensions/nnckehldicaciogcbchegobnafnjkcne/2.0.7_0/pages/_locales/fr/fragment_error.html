Vous continuez de rencontrer des problèmes ?

Pour que vous puissiez utiliser l'extension Tonalité Google, le son émis doit être détecté via celle-ci. Il est possible que le micro de votre ordinateur soit désactivé. Vérifiez les paramètres de votre micro.

Tonalité Google est compatible avec la plupart des ordinateurs, mais il est possible qu'elle ne fonctionne pas sur certains en raison de problèmes liés au matériel. Si tel est votre cas, indiquez-le-nous.
