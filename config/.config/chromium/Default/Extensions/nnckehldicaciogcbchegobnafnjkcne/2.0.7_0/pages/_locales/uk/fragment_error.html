Проблему не вирішено?

Для належної роботи Google Tone має розпізнавати власні сигнали. Можливо, мікрофон комп’ютера вимкнено. Перевірте налаштування мікрофона.

Більшість комп’ютерів підтримують Google Tone, однак на деяких пристроях він може не працювати через проблеми з обладнанням. Зверніться до нас, якщо Google Tone не працює на вашому комп’ютері.
