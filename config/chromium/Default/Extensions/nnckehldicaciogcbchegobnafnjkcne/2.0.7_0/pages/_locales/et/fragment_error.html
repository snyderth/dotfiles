Kas see ikka ei tööta?

Google Tone peab korralikult töötamiseks end kuulma. Teie arvuti mikrofon võib olla välja lülitatud. Kontrollige mikrofoniseadeid.

Google Tone'i toetatakse enamikus arvutites, kuid see ei pruugi riistvara probleemide tõttu mõnes arvutis töötada. Kui see teil ei tööta, andke meile sellest teada.
