Aínda non funciona?

Para que Google Tone funcione correctamente, ten que escoitarse a si mesmo. É posible que o micrófono do teu ordenador estea desactivado. Comproba a configuración do micrófono.

Google Tone é compatible coa maioría dos ordenadores, pero quizais non funcione nalgúns debido a problemas de hardware. Se non che funciona, contacta con nós.
