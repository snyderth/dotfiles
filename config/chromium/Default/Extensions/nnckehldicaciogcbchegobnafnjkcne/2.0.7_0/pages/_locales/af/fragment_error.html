Werk dit steeds nie?

Google Toon moet homself hoor om behoorlik te werk. Jou rekenaar se mikrofoon is dalk afgeskakel. Gaan jou mikrofooninstellings na.

Google Toon word op die meeste rekenaars gesteun, maar op sommige sal dit dalk weens hardewarekwessies nie werk nie. Laat ons weet as dit nie vir jou werk nie.
